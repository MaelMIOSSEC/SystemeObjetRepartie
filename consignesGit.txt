1 - Génère une clé SSH : ssh-keygen -t ed25519 -C "mama.miossec@gmail.com"
2 - Copier la clé publique : cat ~/.ssh/id_ed25519.pub
3 - L'ajouter sur GitHub : Settings > SSH and GPG keys > New SSH key et coller le texte.
4 - Ouvrir un terminal dans mon dossier de travail habituel.
5 - Taper : git clone git@github.com:MaelMIOSSEC/SystemeObjetRepartie.git
6 - Exécuter npm install pour récupérer les fichiers du dossier node_module.

Pour chaque modification, à la racine du projet : 
	1 - git status
	2 - git add .
	3 - git commit -m "message du commit"
	4 - git push
